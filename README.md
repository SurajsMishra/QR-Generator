# QR Generator

A simple and user-friendly **QR Code Generator** built using Python. This application allows users to generate QR codes from text or URLs and save them as image files.

---

## 🚀 Features

* Generate QR codes from any text or URL
* Simple graphical user interface (GUI)
* Save QR codes as `.png` images
* Lightweight and easy to use

---

## 🛠️ Technologies Used

* Python
* `qrcode` library
* `tkinter` (for GUI)

---

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/qr-generator.git
   cd qr-generator
   ```

2. Install required dependencies:

   ```bash
   pip install qrcode[pil]
   ```

---

## ▶️ Usage

Run the Python file:

```bash
python qr.py
```

### Steps:

1. Enter the text or URL in the input field
2. Click on **"Generate QR Code"**
3. Click on **"Save QR Code"**
4. Choose location and save the image
